• git clone копируем репозиторий
  • Перед началом работы нужно выполнить некоторые настройки:
    git config --global user.name "Your Name" # указать имя, которым будут подписаны коммиты
    git config --global user.email "e@w.com" # указать электропочту, которая будет в описании коммитера
  • сначала добавляем файл в индекс (git add)
  • git commit -am "added new file index.html" - это фиксируем и «сохраняем» (git commit)
  • git push - отправить все закомеченные изменения в репозиторий на сервер
  • git pull - забрать свежее обновления
  • git add - добавить файлы для отслеживания
